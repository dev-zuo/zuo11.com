# 0-9数字中，每个数字的宽度是多少

什么数字最宽，以字体PingFangSC-Regular，大小12px为例

```bash
# 测试10个x width宽度
10个0 # 72
10个1 # 48.13
10个2 # 72
10个3 # 72
10个4 # 72
10个5 # 72
10个6 # 72
10个7 # 65.64
10个8 # 72
10个9 # 72
```
总结，一般宽度 1 < 7 < 0,2,3,4,5,6,8,9  但不排除其他字体没有差异，同理我们可以使用这种方法看26个字母里每个字母的占用宽度为多少
