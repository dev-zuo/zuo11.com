
# 常用的.gitignore配置

一般用git做代码管理时，需要将某些系统文件或配置文件不加入版本控制，就需要在.gitignore文件里加入对应的文件名或路径，下面是一个常用的.gitignore配置

```bash
.DS_Store    # 苹果隐藏文件
node_modules # node包
/dist        # dist目录

# local env files
.env.local
.env.*.local

# Log files
npm-debug.log*
yarn-debug.log*
yarn-error.log*

# Editor directories and files
.idea
.vscode
*.suo
*.ntvs*
*.njsproj
*.sln
*.sw*

package-lock.json
```
