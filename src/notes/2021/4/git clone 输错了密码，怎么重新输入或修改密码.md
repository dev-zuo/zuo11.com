# git clone 输错了密码，怎么重新输入或修改密码

在 windows 系统中，如果是第一次拉取代码，而且是使用的 http 方式，涉及到登录的时候，会要求输入 git 相关账号密码，如果账号密码不小心输入错误。后面每次 git clone 都会提示 auth 失败。

这时我们可以在 **控制面板 - 用户账户 - 凭据管理器** 里修改对应的密码
